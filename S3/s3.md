###  create a bucket

```
aws s3api create-bucket \
    --bucket $bucketname \
    --region $bucketregion \
    --create-bucket-configuration LocationConstraint=$bucketregion
   ```
###  upload file
```
aws s3 cp $filename s3://$bucketname/ --region=$bucketregion
```
